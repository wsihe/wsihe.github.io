Vue2.0多页签应用(2)-设置

2017.07.03

前端开发神器-Webstorm

###开始

从 Webstorm 官网下载对应系统的安装包，webstorm 有 30 天的免费试用期，注册机资源

###主题

推荐使用 Webstorm 黑色的主题，既保护眼睛也会让你更加的专注书写代码

  • Win:File > Settings > Appearance & Behavior Mac:Preferences > Appearance

###配置
合适的配置，编码的时候让人更舒畅

  • Webstorm 配置可以导出导出相应的备份

  • 可以引入Webstorm配置文件，也可以下载相关的配置导入，例如 Monokai-Sublime

  • 选择 Import Settings，导入下载的Monokai-Sublime.jar即可

  • 更多主题配置下载 color-themes

###扩展

  • 安装Vue插件

  • 旧版本的webstorm 要在 File Types: HTML 添加 *.vue类型

  • JavScript：版本设置为 ECMAScript6, [x] Prefer Strict mode

  • 设置代码模版

附加：
把 webstorm 的快捷键放在这里是方便自己去记忆，方便快捷的使用webstorm

###基本

编辑窗口的常用操作

  • 撤销，Undo， Win:Ctrl + Z; Mac:Com + Z

  • 恢复，Redo， Win:Ctrl + Shift + Z; Mac:Com + Shift + Z

  • 查找，Find， Win:Ctrl + F; Mac:Com + F

  • 在某一文件夹下查找内容，Find in Path， Win:Ctrl + Shift + F; Mac:Ctrl + Shift + F

  • 替换，Replace， Win:Ctrl + R; Mac:Com + R，

  • 格式化代码，Reformat Code， Win:Ctrl + Alt + L， Mac:Option + Com + L

对当前光标所在行 或者 选中部分做操作

  • 向下复制一行或一个代码块，Duplicate Line or Block， Win:Ctrl + D; Mac: Com + D

  • 剪切，Cut， Win:Ctrl + X; Mac: Com + X

  • 删除，Delete Line， Win:Ctrl + Y; Mac: Com + Y

  • 上下移动行，Move Line Up/Down， Win:Alt + Shift + 上/下箭头;Mac:Option + Shift + 上/下箭头

  • 上下移动代码块，Move Statement Up/Down， Ctrl + Shift + 上/下箭头

###操作编辑窗体

  • 关闭当前的编辑窗体，Close Active Tab， Win:Ctrl + F4; Mac:Com + F4 或 Com + W

  • 左右切换编辑窗体，Select Previous/Next Tab， Win:Alt + 左右箭头; Mac:Ctrl + 左右箭头，

  • 最大化编辑窗体(隐藏所有工具窗口)，Hide All Toll Windows Win:Ctrl + Shift + F12; Mac:Com + Shift + F12

###光标移动

  • 前后跳单词，Move Caret to Previous/Next Word Win:Ctrl + 左右箭头; Mac:Option + 左右键

  • 跳到行首行尾，Move Caret to Line Start/End Win:Home/End; Mac:Com + 左右箭头

  • 在下一行书写，Start Next Line Win:Shift + Enter; Mac:Shift + Enter

  • 在上一行书写，Start New Line Before Current， Win:Ctrl + Alt + Enter; Mac:Option + Com + Enter

  • 跳到指定行， Win:Ctrl + G; Mac:Com + G

###折叠代码

  • 折叠当前代码块，Collapse， Win:Ctrl + Num-; Mac:Com + Num-

  • 展开当前代码块，Expand， Win:Ctrl + Num+; Mac:Com + Num+

  • 折叠全部代码块，Collapse All， Win:Ctrl + Shift + Num-; Mac:Com + Shift + Num-

  • 展开全部代码块，Expand All， Win:Ctrl + Shift + Num+; Mac:Com + Shift + Num+

###选中

  • 选中的超级扩展，Extend Selection， Win:Ctrl + W; Mac:Ctrl + W，

  • 重命名，Rename， Win:Shift + F6; Mac:Shift + F6，

###粘贴

  • 粘贴，Paste， Win:Ctrl + V; Mac:Com + V

  • 从复制历史中粘贴，Paste form History， Win:Ctrl + Shift + V; Mac:Com + Shift + V

###文件

  • 打开文件，File… Win:Ctrl + Shift + N; Mac:Com + Shift + N

  • 保存全部，Save All， Win:Ctrl + S; Mac:Com + S

  • 新建文件，New… Win:Alt + N; Mac:Control + N，在当前文件同级或者选中的文件夹下

  • 打开最近打开的文件，Recent Files， Mac:Com + E，会给出一个最近打开的文件列表

  • 复制文件路径，Copy Paths， Mac: Com + Shift + C

###面板之间的切换

  • 打开/收起命令行面板，Terminal， Mac: Option + F12

  • 左边的文件栏，Send to Left, Project Mac: Com + 1

###单项

  • 两行变一行，Join Lines， Mac: Control + Shift + J
