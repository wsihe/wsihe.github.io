Vue2.0 UI 组件库开源项目 - Wuui

2017.08.21

1.这是怎样的一个项目？

Wuui 是基于 Ant Design 的 Vue2.0 组件实现，开发和服务于 PC 中后台产品。

2.已经有优秀的 Vue 开源组件，为什么还要重复造轮子？

最早想写一个 Vue UI 库是因为看到 Element，里面各种 UI 组件的实现，而在公司项目用 Vue1.0 也写了部分UI组件，慢慢萌生自己写一套 Vue2.0 组件来用的想法，在造轮子过程中的理解 Vue2.0 各项细节，而且自己锻造的兵器，用起来更加趁手。

3.想做成怎样，实现哪些的功能？

自定义主题，多功能多样组件，按需引入…

4.希望？

当然希望能有更多朋友一起参与进来，贡献代码！
