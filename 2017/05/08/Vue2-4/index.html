在vue-cli(1)，已经详细描述了搭建开发环境的配置，接着下来说一下的生产环境配置
执行 npm run build 的时候，引入的关系：
npm run dev => build.js => webpack.prod.conf.js(index.js) => webpack.base.conf.js

build.js

当我们执行 npm run build 时运行的是 node build/build.js，该文件主要做了以下事情。

  • 引入相关配置
