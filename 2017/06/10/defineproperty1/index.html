JavaScript 实现监听一个对象的变化

2017.06.10

知识点：当对象的某个属性变化的时候，可以通过 ES5 中的 Object.defineProperty 方法触发其中的 getter 和 setter 函数，执行自定义的回调函数 。

Object.defineProperty

简单通过 Object.defineProperty 给一个对象定义一个属性：

  • 读取属性值的时候会调用 get 方法
  • 设置属性值的时候会调用 set 方法

对象往往是一个深层次的结构，我们要做的是遍历对象的属性，把对象的属性都使用 Object.defineProperty 转为 getter/setter，当我们修改对象值的时候，调用set方法执行回调通知。
