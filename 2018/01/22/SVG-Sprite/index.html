SVG Sprite 技术实践

2018.01.22

背景

在项目中我们经常会遇到使用多个icon的情况，为了减少网页的http请求，提高页面的性能，通常的做是会把icon合并到一个文件中，再使用相应的技术把所需的图形从icon集合文件中取出并显示。目前使用的比较多的应该就是 CSS Sprite和 Icon Font。

CSS Sprite

CSS Sprite 在国内很多人叫css精灵，其原理是把网页中一些背景图片整合到一张图片文件中，再利用CSS的background-image，background- repeat，background-position的组合进行背景定位，background-position可以用数字精确的定位出背景图片的位置。

Icon Font

Icon Font 将矢量的图标做成字体，一次网络请求就够了，可以保真缩放
